<h1 align="center" id="title">Shortlink</h1>

<p id="description">URL shortener project</p>

<h2>🛠️ Installation Steps:</h2>

```
curl -fsSL https://kool.dev/install | bash
```

```
kool self-update
```

```
kool run setup
```

```
kool start
```
